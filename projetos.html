Meus Projetos

Aqui vocês encontrarão alguns dos projetos que venho desenvolvendo, com detalhes sobre cada um deles.

Cofrinho Digital - Uninter

Projeto acadêmico da disciplina de Programação Orientada a Objetos. Trata-se de um cofrinho digital com funcionalidades para adicionar, remover e consultar moedas, além de converter para reais.

Desafio Conta Terminal - DIO Java Básico

Aplicação em Java que simula a abertura de uma conta bancária, praticando o uso de Scanner, entrada/saída de dados e concatenação de strings.

Banco Projeto - DIO

Sistema bancário simples feito com Java, aplicando modelagem de classes, listas, abstração e interfaces. Projeto prático para consolidar POO.
